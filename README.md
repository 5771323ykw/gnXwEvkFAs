# 项目简介

## 核心功能
- 登录注册
- 全文检索
- 商品评论
- 用户中心
- 购物车
- 商品下单
- 消息中心
- 卖家管理

## 适用场景
- 二手商品交易
- 在线交易平台搭建
- 电商平台后端开发

## 技术特点
- 后端：使用 Python 语言和 Django 框架进行开发，确保代码的稳定性和可维护性。
- 前端：采用 HTML、CSS、JavaScript、jQuery 等技术实现用户界面和交互。
- 数据库：使用 MySQL 数据库进行数据存储和管理。
- 开发环境：支持 Python3.0 以上版本，MySQL8.0 以上版本。
- 工具：使用 Pycharm 进行代码编写，navicat 进行数据库管理。
